Eco Italia —  Full Stack Shop

React + Node.js (без MongoDB, дані в пам'яті покищо)



Backend
```bash
cd backend
npm install
npm run dev   # http://localhost:5000
```

Frontend 
```bash
cd frontend
npm install
npm start     # http://localhost:3000
```
